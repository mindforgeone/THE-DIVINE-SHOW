# 120 дней: форма, 1С, рынок

Светлый интерактивный трекер на 120 дней для идеальной формы, маниакального роста в 1С и сильного выхода на рынок.

## Что внутри

- старт марафона с текущей даты;
- три результата дня: поддержание, рост, прорыв;
- XP, уровень, серия, карта 120 дней;
- графики веса, калорий, времени и траектории 1С/рынок;
- недельные итоги и чекпоинты 7 / 14 / 30 / 60 / 90 / 120;
- экспорт Markdown и JSON для анализа нейросетью.

## Запуск

```bash
npm install
npm run dev
```

## Проверка

```bash
npm run lint
npm run build
```
